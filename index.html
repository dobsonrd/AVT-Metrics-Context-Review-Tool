v1.0
Initial build — 58 Tier 1 metrics
Initial tool built for the AVT Assurance deepdive. 58 Tier 1 metrics from the AVT Metrics Taxonomy (danjscho/avt-metrics-taxonomy v5.5), organised into 7 thematic buckets. Each metric has a risk narrative, harm pathway, assurance question, measurement method, trigger conditions, and phase note. Decision capture fields for agreed measurement, threshold, owner, and parking lot notes. Bucket overview tab with editable definitions and rationale. Export tab generating a plain-text decisions log.
Taxonomy source: AVT Metrics Taxonomy by danjscho — github.com/danjscho/avt-metrics-taxonomy · docs site
Prototype for discussion — not an NHS-endorsed standard. Tier assignments and threshold numbers are calibrated starting points requiring local review before contractual use.
